曲タイトル部分の画像範囲指定
  left-top		左上X座標
  left-bottom	左上X座標から下に何ピクセル分
  right-top		右上X座標
  right-bottom	右上X座標から下に何ピクセル分

曲タイトルリスト
  music_list_filename	ファイル名（プログラムソースと同じフォルダに配置）

ワーク用ファイル
  capture_filename			画面キャプチャ保存ファイル名
  capture_clip_filename		画面キャプチャ保存ファイルからタイトル部分切り出しファイル名

判定結果テキストファイル
  title_text_filename		結果ファイル名（曲名がテキスト出力される）
